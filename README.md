# 🎮 Aplex Runner Game

A simple and fun 2D browser game built using HTML, CSS, and JavaScript. In this game, the player must avoid obstacles, collect coins, and survive as long as possible.

## ⚙ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/aplex-runner-game.git
```

### 2️⃣ Open Project Folder

```bash
cd aplex-runner-game
```

### 3️⃣ Run the Game

Simply open the `index.html` file in your browser.

```bash
start index.html
```

---

## 🎮 How to Play

| Key | Action |
|------|---------|
| ⬅ Left Arrow | Move Left |
| ➡ Right Arrow | Move Right |
| ⬆ Up Arrow | Jump |
| P | Pause Game |

---

## 🏆 Game Rules

- Avoid hitting obstacles
- Collect coins to increase score
- Survive as long as possible
- Game ends when player hits an obstacle
